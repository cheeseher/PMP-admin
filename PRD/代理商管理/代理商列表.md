# 代理商列表

## 页面描述
用于展示和管理系统中的代理商信息。

## 字段说明

| 字段名 | 含义 | 类型 | 必填 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| agentId | 代理商 ID | String | 是 | 唯一标识 |
| username | 账号 | String | 是 | 登录账号 |
| nickname | 昵称 | String | 是 | 显示名称 |
| parentAgent | 上级 | String | 否 | 上级代理商名称，无上级显示"-" |
| subAccountCount | 子账户数量 | Number | 否 | 下级子账户的数量 |
| merchantCount | 商户数量 | Number | 否 | 关联的商户数量 |
| balance | 余额 | String | 否 | 账户总余额 |
| availableBalance | 可用 | String | 否 | 可用余额 |
| frozenBalance | 冻结 | String | 否 | 冻结金额 |
| rate | 费率 | String | 是 | 代理费率，如 "1.2%" |
| lastLoginIp | 上次登录 IP | String | 否 | 最后一次登录的 IP 地址 |
| loginCount | 登录次数 | Number | 否 | 总登录次数 |
| googleAuth | 谷歌认证 | Boolean | 是 | 是否开启谷歌验证，true/false |
| oneClickLogin | 一键登录 | Boolean | 是 | 是否允许一键登录，true/false |
| role | 角色 | String | 是 | 代理商角色，如"总代理"、"普通代理" |
| status | 状态 | String | 是 | 账户状态，"active"(正常) / "disabled"(禁用) |

## 交互说明

### 筛选区
- **代理商 ID**：精确搜索
- **账号**：模糊搜索
- **状态**：下拉选择（全部、正常、禁用）

### 操作区
- **新增代理商**：点击弹出新增弹窗
- **导出**：导出当前列表数据

### 表格区
- **固定列**：代理商 ID（左侧固定）、操作（右侧固定）
- **状态展示**：使用 Tag 组件展示状态和谷歌认证状态
- **开关展示**：一键登录使用 Switch 组件展示（禁用状态）

### 操作列
- **查看**：查看代理商详情
- **编辑**：编辑代理商信息
- **删除**：删除代理商（需二次确认）
