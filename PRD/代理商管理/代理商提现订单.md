# 代理商提现订单

## 页面描述
用于展示和审核代理商的提现申请。

## 字段说明

| 字段名 | 含义 | 类型 | 必填 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| withdrawId | 提现单号 | String | 是 | 唯一标识 |
| agentName | 代理商名称 | String | 是 | 申请提现的代理商 |
| amount | 提现金额 | Number | 是 | 申请金额 |
| status | 状态 | String | 是 | 待审核/已通过/已拒绝 |
| createTime | 申请时间 | String | 是 | 申请提交时间 |
| auditTime | 审核时间 | String | 否 | 完成审核的时间 |
| remark | 备注 | String | 否 | 申请或审核备注 |

## 交互说明

### 筛选区
- **提现单号**：精确搜索
- **代理商名称**：模糊搜索
- **状态**：下拉选择（全部、待审核、已通过、已拒绝）

### 操作区
- **导出**：导出当前列表数据

### 表格区
- **操作列**：
  - **审核**：仅“待审核”状态显示，点击弹出审核弹窗
  - **查看**：查看详情
