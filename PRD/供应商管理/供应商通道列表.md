# 供应商通道列表

## 页面概述
管理供应商通道信息，支持查看、新增、编辑、删除及批量操作。

## 主要功能
- 供应商通道信息展示
- 多条件筛选查询
- 新增/编辑通道
- 批量删除
- 批量设置分组
- 拉单测试

## 字段说明

### 表格字段
| 字段名 | 字段含义 | 说明 |
|--------|----------|------|
| 供应商通道ID | 通道唯一标识 | 数字类型 |
| 通道名称 | 通道显示名称 | 字符串 |
| 通道编码 | 通道唯一编码 | 字符串 |
| 通道费率 | 当前费率及定时费率 | 显示当前费率，如有定时生效费率则展示生效时间 |
| 交易金额区间 | 单笔交易限额 | 最小金额 - 最大金额 |
| 分组 | 通道分组 | 用于聚合管理 |
| 状态 | 启用/禁用 | 开关状态 |
| 备注 | 备注信息 | 文本描述 |

### 筛选字段
| 字段名 | 筛选类型 | 说明 |
|--------|----------|------|
| 供应商通道ID | 精准匹配 | 输入框，宽度 168px |
| 通道名称 | 模糊搜索 | 输入框，宽度 168px |
| 通道编码 | 模糊搜索 | 输入框，宽度 168px |
| 分组 | 下拉选择 | 宽度 168px |
| 状态 | 下拉选择 | 宽度 168px，选项：全部/启用/禁用 |

## 界面优化
- 筛选区布局优化：采用流式 Flex 排布，统一控件宽度为 168px，解决横向滚动问题。
