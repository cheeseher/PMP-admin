# 设计规范

## 1. 概览
暂无详细规范。请在此维护 UI/UX 规范，防止样式和交互逻辑偏差。

## 2. 规范列表
- 组件层级
- 布局规则
- 配色/字体/间距 Token
- 交互动画逻辑
- 供应商通道 (`Channel.vue`) 表单新增“支持设备”的多选框控件。顶部搜索栏支持单选筛选，表格使用 `el-tag` 不同的类型（info/success/warning）做视觉区分展示所见设备。
- 商户收款统计 (`MerchantIncome.vue`) 页面数据表格列名及数量更新，新增4个业务相关金额列（隔日修改订单金额、手续费、冲账金额及调账后入账金额），这4个列均使用 `amount-cell` 样式以保持等宽数字排版与金额的精确对齐显示。
- **支付产品管理 (`Product.vue`)**：编辑弹窗中的“商户费率”同步功能已优化。同步勾选框从输入框旁移至底部的“同步配置到商户”下方。仅当选择“立即同步”或“定时同步”时，才会展示“同步商户费率”勾选框。**默认不勾选**，由操作员手动确认是否同步费率。
