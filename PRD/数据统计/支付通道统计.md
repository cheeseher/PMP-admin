# 支付通道统计

## 页面概述
统计分析支付通道的交易数据，帮助运营人员了解通道的成功率、成本和交易量。

## 主要功能
- 核心指标卡片展示（总成功收款金额、总通道成本、总成功单数/总笔数）
- 多维度筛选查询
- 数据列表展示
- 数据导出

## 字段说明

### 统计卡片
| 指标名 | 说明 |
|--------|------|
| 总成功收款金额 | 所有筛选记录的成功收款总额 |
| 总通道成本 | 所有筛选记录的通道手续费总额 |
| 总成功单数/总笔数 | 成功订单数量 / 总订单数量 |

### 筛选字段
| 字段名 | 筛选类型 | 说明 |
|--------|----------|------|
| 时间筛选 | 时间范围 | 默认为空，支持快捷选项 |
| 供应商通道ID | 精准匹配 | 输入框，宽度 168px |
| 通道名称 | 模糊搜索 | 输入框，宽度 168px |
| 通道编码 | 精准匹配 | 输入框，宽度 168px |

### 表格字段
| 字段名 | 字段含义 | 说明 |
|--------|----------|------|
| 供应商通道ID | 通道唯一标识 | 数字类型 |
| 通道名称 | 供应商名称 \| 通道名称 | 组合显示 |
| 通道编码 | 通道唯一编码 | 字符串 |
| 成功笔数 | 交易成功的订单数量 | 数字 |
| 总笔数 | 总订单数量 | 数字 |
| 成功收款金额 | 实际到账金额 | 金额格式 |
| 成功率 | 成功笔数 / 总笔数 | 百分比格式 |
| 通道成本 | 通道手续费 | 金额格式 |

## 界面优化
- 筛选区布局优化：采用流式 Flex 排布，统一控件宽度为 168px，解决横向滚动问题。
- 统计卡片：适配不同屏幕尺寸，响应式布局。
