# 商户提现审核

## 页面功能
展示和管理商户提现申请列表，支持审核操作和数据导出。

## 页面结构
- 筛选区域：商户名称、提现单号、状态、申请时间
- 操作区域：批量审核（导出按钮位于表格工具栏，紧邻批量审核右侧）
- 表格区域：提现申请列表
- 分页区域：数据分页展示

## 字段说明

### 筛选字段
| 字段名 | 字段含义 | 交互类型 | 是否必填 | 备注 |
|--------|----------|----------|----------|------|
| 商户名称 | 选择商户 | 下拉选择 | 否 | 支持搜索，选项：商户A、商户B、商户C |
| 提现单号 | 提现订单号 | 文本输入 | 否 | 支持模糊搜索 |
| 状态 | 提现状态 | 下拉选择 | 否 | 选项：未确认、成功、失败 |
| 申请时间 | 时间范围 | 日期选择 | 否 | 日期范围选择器 |

### 表格字段
| 字段名 | 字段含义 | 数据类型 | 备注 |
|--------|----------|----------|------|
| 商户名称 | 商户名称 | 文本 | 显示商户名称 |
| 提现订单号 | 提现订单号 | 文本 | 唯一标识 |
| 订单号格式 | 订单号生成规则 | 文本 | MTX + yyyyMMddHHmmss + 6位随机数字 |
| 提现数量（U） | 提现USDT数量 | 数值 | 右对齐显示 |
| 类型 | 提现类型 | 标签 | TRX |
| 钱包地址 | 提现钱包地址 | 文本 | 超长显示省略号 |
| 提现金额（CNY） | 提现人民币金额 | 数值 | 右对齐显示，带¥符号 |
| 备注 | 固定模板文案 | 文本 | 根据行数据动态生成，格式：商户【账户名称】申请下发商户余额：XXXXX汇率和数量：{数量}*{汇率}下发 RMB：{金额}下发 地址：{地址}下发申请成功，请耐心等待财务打款。谢谢！（不接受期间更换、修改地址及金额，出现金额损失自行负责哦） |

| 操作时间 | 申请时间 | 时间 | 格式：YYYY-MM-DD HH:mm:ss |
| 操作列 | 操作按钮 | 按钮组 | 审核 |

## 操作说明

### 审核操作
点击【审核】按钮，弹出审核弹窗：
- 显示提现申请基本信息
- 选择审核结果：通过/拒绝
- 输入审核备注
- 点击确定提交审核结果

### 拒绝操作
不提供单独【拒绝】按钮；在审核弹窗中选择“拒绝”，填写审核备注后提交。

### 批量审核
1. 勾选多条提现申请记录
2. 点击【批量审核】按钮
3. 选择审核结果：通过/拒绝
4. 输入审核备注
5. 点击确定批量提交审核结果

### 导出操作
导出按钮位于表格工具栏，紧邻【批量审核】右侧；点击后导出当前筛选条件下的数据。

## 状态说明
- **未确认**：提现申请待审核，显示为橙色标签
- **成功**：提现申请已通过，显示为绿色标签
- **失败**：提现申请已拒绝，显示为红色标签

## 交互规则
1. 页面加载时自动查询数据
2. 筛选条件变化后需手动点击查询按钮
3. 表格支持多选，用于批量审核
4. 审核弹窗关闭时重置表单数据
5. 操作成功后自动刷新表格数据
6. 仅“未确认”状态显示可用的【审核】按钮，其他状态禁用
7. 拒绝不再提供单独按钮，统一在审核弹窗选择“拒绝”并填写审核备注
8. 类型列展示统一为 TRX，避免误显示为其他类型
9. 提现订单号格式为 MTX + yyyyMMddHHmmss + 6位随机数字，示例：MTX20251112153045123456
